# rc-auto

## Motor

- Outer diameter: 36mm
- Mounting holes distance: 25mm, hex pattern
- Shaft teut diameter: 11.5mm

## Motor mount

- Interface with Plate
	- 3 holes, 21.5mm apart, M3
	- Corner hole is 5.25mm from center
	- DriveShaft is 12.5mm diam
	- DriveShaft is 22mm above Plate
